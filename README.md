# AI 单词短文生成器

Anki 插件，利用 AI 将当天待学习的单词生成英语短文，帮助在语境中记忆单词。

## 功能

- 自动获取 Anki 中当天待复习+新学的单词
- 调用 OpenAI 兼容 API（DeepSeek、OpenAI 等）生成自然英语短文
- 支持自动检测单词字段，也可手动指定笔记类型和字段
- 短文后自动按词性分类列出目标单词及释义
- 可选附带中文翻译
- 支持手动触发或进入复习时自动生成
- 短文可保存为本地 TXT 文件

## 安装

1. 从 [Releases](../../releases) 下载 `anki_word_passage.ankiaddon`
2. 打开 Anki → 工具 → 插件 → 从文件安装
3. 选择下载的 `.ankiaddon` 文件，安装后重启 Anki

## 使用

1. **配置 API Key**：打开 Anki → 工具 → AI 单词短文设置，填入 API Key 等必要信息
2. **生成短文**：在主界面点击「生成今日单词短文」按钮，或在 工具 菜单中点击「生成今日单词短文」
3. 插件会自动获取今天需要复习和学习的单词，调用 AI 生成短文
4. 生成的短文可以在弹窗中查看，也可以保存为 TXT 文件

> **提示**：开启「自动生成」后，每次进入复习时会自动生成当天的短文。

## 配置

在 Anki 中通过 **工具 → AI 单词短文设置** 打开配置面板：

| 配置项 | 说明 |
|---|---|
| API 地址 | OpenAI 兼容 API 地址，如 `https://api.deepseek.com/v1` |
| API Key | 你的 API 密钥 |
| 模型 | 模型名称，如 `deepseek-chat` |
| 单词字段 | 笔记中存放单词的字段名（留空自动检测） |
| 笔记类型 | 筛选指定笔记类型（留空则不限） |
| 每日单词上限 | 最多取多少个单词生成短文 |
| 每篇单词数 | 每篇短文包含的目标单词数 |
| 短文词数 | 每篇短文的输出词数 |
| 附带翻译 | 是否在短文后附加中文翻译 |
| 词性分类 | 是否按词性对目标单词分组列出 |
| 自动生成 | 进入复习时是否自动生成短文 |
| 保存路径 | TXT 文件保存目录（按日期自动分文件夹） |

## 依赖

- Anki >= 2.1.50（自带 Python 运行环境，无需额外安装）
- 一个 OpenAI 兼容 API 的密钥
